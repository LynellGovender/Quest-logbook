# Quest Logbook 📖

A simple web-based quest tracking app where users can create, manage, and complete personal quests or goals.

## 🚀 Features
- Add new quests with title and description  
- Track progress or status of each quest  
- Mark quests as completed  
- Simple and clean user interface  
- Works directly in the browser  

## 🛠️ Built With
- HTML  
- CSS  
- JavaScript
